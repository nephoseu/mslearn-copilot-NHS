# Explore Microsoft 365 Copilot (NHS Scenario)

Welcome to the world of **Microsoft 365 Copilot** in a realistic **NHS context**.

In this exercise, you will use Copilot to explore and refine a **service improvement idea** at a fictional organisation: **Northbridge NHS Foundation Trust**.

Northbridge NHS Foundation Trust is experiencing sustained pressure from increasing elective demand, constrained capacity, and growing administrative workload. Clinical and operational staff spend significant time drafting routine communications, summarising meetings, producing reports, and preparing board papers.

Using Microsoft 365 Copilot, you will explore how generative AI can support **non-clinical productivity tasks** such as:
- Drafting and refining operational documents
- Researching and structuring improvement initiatives
- Creating business cases and briefing materials
- Visualising benefit assumptions
- Preparing leadership communications

By the end of this exercise, you will have produced a **coherent set of NHS-appropriate materials** that could realistically be used to sponsor a small pilot initiative.

> **Important**  
> This exercise provides example prompts you can use with Copilot. Use them as a *starting point*. You are encouraged to modify prompts and iterate.

This exercise should take approximately **40 minutes** to complete.

> **Note**  
> This exercise requires a **Microsoft 365 Copilot** license.

> **Safety and Governance Notice**  
> - Fictional training scenario  
> - No patient-identifiable information (PII)  
> - No clinical decision-making  
> - Human review required for all outputs

## Use Copilot to explore a document and research an idea

1. Download **Business Idea.docx** from the training repository.
2. Upload it to **OneDrive** and open it in **Microsoft Word**.
3. Open the **Copilot** pane and run:

```prompt
Summarize this document into 5 key points, and suggest next steps.
```

4. Ask a follow-up question:

```prompt
What are the key operational risks and constraints in this proposal?
```

5. Research pilot setup steps:

```prompt
What are the typical steps to start a productivity pilot using Microsoft 365 Copilot in an NHS Trust?
Answer with a numbered list.
```

6. Paste the list into the document and ask Copilot to **visualize it as a table**. Add columns for **Owner** and **References**.

## Use Copilot to create a pilot business case

1. Ask Copilot:

```prompt
Suggest 10 professional programme names suitable for an NHS service improvement pilot.
```

2. Create a new document and draft a business case:

```prompt
Write a pilot business case for the NHS programme "Outpatient Flow Accelerator" based on /Business Idea.docx.
Include executive summary, scope, governance constraints, success measures, risks, and next steps.
```

3. Refine for board audience:

```prompt
Rewrite this for an NHS Trust Board audience, focusing on assurance and risk.
```

Save as **Pilot Business Case.docx**.

---

## Visualize benefit assumptions in Excel

1. In Word, ask Copilot:

```prompt
Create a simple 5-year benefit model for an NHS productivity pilot using reasonable assumptions.
```

2. Copy the table into **Excel**, format as table, and ask:

```prompt
Visualize the cost avoidance trend as a line chart.
```

Save as **Benefits Model.xlsx**.

## Create a board briefing presentation

1. Create a new PowerPoint and ask Copilot:

```prompt
Create a presentation for an NHS Trust Board about a 12-week pilot called "Outpatient Flow Accelerator".
Include challenge, scope, governance, benefits, risks, and approvals needed.
```

2. Add a slide:

```prompt
Add a slide titled "Information Governance and Safety".
```

Save as **NHS Copilot Pilot Briefing.pptx**.

## Arrange a leadership meeting

1. In Outlook Copilot:

```
What events do I have scheduled this week?
```

2. Draft an email:

```prompt
Write a concise email to the Director of Operations requesting sponsorship for the Outpatient Flow Accelerator pilot.
Use a professional NHS tone.
```
